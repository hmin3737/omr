# OMR 통계 생성기

OMR 스캔 채점결과 파일(xls/xlsx/csv)을 올리면 시험 통계 자료를 **한 페이지짜리 xlsx · pdf**로 생성하는 웹앱입니다. 모든 처리는 브라우저 안에서 이뤄지며 데이터는 서버로 전송되지 않습니다.

## 입력

1. **시험명** (필수)
2. **채점결과 파일** (필수) — `성명 / 수험번호 / 선택 / 점수 / 문항…` 구조, 문항 칸 `O`=정답
3. **허수 표본 제거 기준** (선택) — 입력 점수 **이하** 응시자 제외
4. **원하는 통계 자료** — 응시자수 · 평균 · 표준편차 · 100점 수 · 상위 30% 평균 · 무보정 백분위/표준점수/등급
5. **정답률 낮은 문제 기준(%)** (필수)

## 계산 규칙

- 모든 통계는 **점수가 산출된 응시자**만 대상 (미응시·미채점, 허수 제거분 제외).
- **평균/표준편차**: 모표준편차 사용.
- **100점 수**: 0명이면 0명으로 표기하고 최고점과 그 인원을 함께 기재.
- **상위 30% 평균**: 점수 내림차순 상위 30%(인원 반올림) 평균.
- 수능(대학수학능력시험) 준용 (국어·수학 기준: 평균 100, 표준편차 20):
  - 표준점수 = (원점수 − 평균) / 표준편차 × 20 + 100
  - 백분위 = (본인보다 낮은 점수 인원 + 동점자 수 ÷ 2) / 전체 인원 × 100
  - 등급 = 상위 누적비율 컷 4/11/23/40/60/77/89/96/100%, 동점자는 상위 등급으로 처리
  - 무보정 백분위/표준점수/등급은 **원점수 → 환산표** 형태로 출력
- 모든 소수점은 **소수 셋째 자리에서 반올림**.
